## 🚀 **Getting Started**

### **Prerequisites**
- Node.js (version 14 or higher)
- npm or yarn package manager

### **Installation**

1. **Clone the repository**
```bash
git clone https://github.com/shahrob/Portfolio.git
cd Portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm start
```

4. **Build for production**
```bash
npm run build
```
